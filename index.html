Getting started

  1. Make sure you have minecraft installed or follow guide "Installing Minecraft Java"
  2. Make sure you have forge installed or follow guide "Installing Forge"
  3. Make sure to have the bare minimum mods installed or follow guide "Getting the required mods"
  4. [Optional] Make sure to have the recommended resource packs or follow guide "Getting the recommended resource packs"

Installing Minecraft Java

  1. Download Minecraft from minecraft.net or Microsoft Store App
  2. Open the Minecraft Launcher app
  3. Sign in with your Xbox account
  4. Select Minecraft Java Edition
  5. Click the "Installations" section
  6. Choose "New Installation"
  7. Configure the installation
    • Name: <any name you choose>
    • Version: 1.20.1 (release)
    • Game directory: leave as default
    • Resolution: leave as default

  8. Click "Create" and wait for installation
  9. Go back to the main Minecraft Java page
  10. Choose version 1.20.1 from the dropdown
  11. Click "Play" to load the game
  12. Exit the game and launcher once loaded

Installing Forge

  1. Download Forge installer from: https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html
  2. Open the Forge installer (.jar file)
  3. Accept any security warnings
  4. Install for "Client" (default option)
  5. Go back to Minecraft Launcher
  6. Choose the new "Forge 1.20.1" profile from the dropdown
  7. Click "Play" to launch
  8. Your game is now ready for mods!
