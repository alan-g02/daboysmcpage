Installing CurseForge

  1. Make sure you have minecraft installed and opened at least once or follow the guide. Installing Minecraft
  2. Download and install CurseForge from https://www.curseforge.com/download/app
  3. Open CurseForge.
  4. Select continue as guest.
  5. Go to Home.
  6. Choose the Minecraft Game.
  7. Select standard modding folder.
  8. Click create custom profile.
  9. Configure profile.
    1. Name: Any
    2. Version: 1.20.1
    3. Game type: Forge
    4. Mod loader version: Latest
  10. Click create profile and wait for installation.
  11. Go to settings.
  12. On the game specific section, select minecraft.
  13. Link CurseForge your xbox/microsoft account by clicking Log In
  14. Click Play. This will launch minecraft with the forge launcher. From now on, to join the server launch CurseForge.
