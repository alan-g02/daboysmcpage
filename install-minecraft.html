Installing Minecraft Java

  1. Download Minecraft from https://www.minecraft.net/en-us/download or Microsoft Store App
  2. Open the Minecraft Launcher app
  3. Sign in with your Xbox account
  4. Select Minecraft Java Edition
  5. Click the "Installations" section
  6. Choose "New Installation"
  7. Configure the installation
    • Name: <any name you choose>
    • Version: 1.20.1 (release)
    • Game directory: leave as default
    • Resolution: leave as default

  8. Click "Create" and wait for installation
  9. Go back to the main Minecraft Java page
  10. Choose version 1.20.1 from the dropdown
  11. Click "Play" to load the game
  12. Exit the game and launcher once loaded
