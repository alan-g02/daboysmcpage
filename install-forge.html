Installing Forge

  1. For mac: Install the Oracle JDK
    1. For intel based macs: https://download.oracle.com/java/24/latest/jdk-24_macos-x64_bin.tar.gz
    2. For silicon based macs: https://download.oracle.com/java/24/latest/jdk-24_macos-aarch64_bin.dmg
  2. Download Forge installer from: https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html *! There will be an ads page, wait for 5 seconds and click skip!*
  3. Open the Forge installer (.jar file)
    1. Accept any security warnings
    2. For mac: If apple denies opening:
      1. Open System Settings
      2. Go to privacy and security
      3. Go to the security section
      4. If you opened the file, there will be a message, "forge-1.20...taller.jar" was blocked to protect your Mac.
      5. Click Open Anyway and authorize warnings.
  4. Install for "Client" (default option)
  5. Go back to Minecraft Launcher
  6. Choose the new "Forge 1.20.1" profile from the dropdown
  7. Click "Play" to launch
  8. Your game is now ready for mods!
