CurseForge Mod Install Guide

  1. Open CurseForge
  2. Select the minecraft game
  3. Selected the 1.20.1 profile
  4. Open profile options
  5. Click open folder
  6. Enter to [Profile Name] > mods folder
  7. Add the mod to the folder.
