Installation Instructions Windows

  1. Download all the missing resource packs from the list above
  2. Press Win + R
  3. Type %AppData%
  4. Open .minecraft
  5. Open the resourcepacks folder.
  6. Add the missing .zip files to the folder.

Installation Instructions mac

  1. Download all the missing resource packs from the list above
  2. Open Finder
  3. Press Command + Shift + G. Go > Go to Folder in the menu bar.
  4. Type ~/Library/Application Support/minecraft
  5. Open the resourcepacks folder.
  6. Add the missing .zip files to the folder.
